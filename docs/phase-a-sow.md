# フェーズA SOW（作業範囲）

## 基本機能に含むもの

- 利用者トップ、月間カレンダー、体験詳細
- 3ステップ予約、完了、予約照会、Webキャンセル
- キャンセル待ち登録
- デモ管理ダッシュボード
- 既存3プランからの開催枠作成（単日・毎週最大12枠）、下書き・公開
- 開催枠の編集・複製・履歴なし削除、定員・受付状態変更、開催中止
- 電話予約、予約者・待機者管理、手動繰り上げ
- 4種類の通知プレビュー、失敗・再送サンプル
- 操作履歴、データリセット
- モバイル・PC対応、アクセシビリティ配慮
- 自動テストと基本文書

## フェーズAに含まないもの

- 本物の顧客情報保存
- Firebase Authentication / Firestore
- Vercel Functionsと実運用API
- EmailJSによる外部メール送信
- オンライン決済、LINE通知
- 複数農園、多言語、外部予約サイト連携
- 体験プラン自体の編集、画像アップロード
- 市場検証、営業活動、ココナラ出品作業

## 追加見積にするもの

- 画面・機能・通知種類の追加
- ロゴ、撮影素材、文章の全面制作
- 本番サービスのアカウント開設代行
- データ移行、外部サービス連携
- 公開後の保守、運用監視、問い合わせ対応

追加要望は、内容・工数・納期・価格を提示し、購入者の承認後に着手します。
